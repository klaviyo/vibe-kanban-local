---
title: "Responsible Disclosure"
description: "How to report security issues in this fork"
---

This is a local-only fork of vibe-kanban with no hosted services or
shared infrastructure. There is no bounty program.

If you believe you've found a security issue in the code shipped from
this repository, please open a GitHub issue or contact the maintainer
listed in the repository profile.

For security issues that affect the upstream `BloopAI/vibe-kanban`
hosted product (which has been sunset), refer to the original
upstream policy.
